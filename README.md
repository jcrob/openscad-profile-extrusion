# OpenSCAD Profile Extrusion

Extruded polygon profile from a dimensioned cross-section sketch.

## Files

- `profile_extrusion.scad` — 2D profile + `linear_extrude`

## Usage

1. Open `profile_extrusion.scad` in [OpenSCAD](https://openscad.org/).
2. Adjust `length` (default 50 mm) for extrusion depth.
3. Render (F6) and export STL/OBJ as needed.

## Profile dimensions (mm)

| Feature | Value |
|---|---|
| Top width | 13.8 |
| Overall height | 26.8 |
| Top thickness | 1.3 |
| Left flange depth | 11.0 |
| Right segment width | 7.4 |
| Right notch depth | 2.4 |
| Stem tip width | 0.95 |
